# 📅 Kalender Keluarga

Aplikasi kalender keluarga berbasis web yang menampilkan tiga sistem
penanggalan sekaligus: Masehi, Hijriyah, dan Weton Jawa.

## ✨ Fitur
- Kalender Masehi + Hijriyah + Weton Jawa dalam satu tampilan
- Pengingat ulang tahun Masehi dan Hijriyah
- Penanda weton anggota keluarga (setiap 35 hari)
- Konversi tanggal (Masehi → Hijriyah + Weton + Neptu)
- Ekspor kartu tanggal sebagai gambar PNG
- Backup & restore data keluarga via file JSON
- Tema Cahaya Pagi dan Gelap
- Koreksi Hijriyah manual (sesuai ketetapan Kemenag)

## 🌐 Akses Aplikasi
https://kalenderkeluarga.web.id

## 🛠️ Teknologi
HTML · CSS · JavaScript murni (tanpa framework)
Semua data tersimpan lokal di perangkat pengguna (localStorage)
